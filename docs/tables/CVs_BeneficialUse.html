Table: BeneficialUse

Beneficial uses associated with the site (e.g. irrigation, municipal, commercial, thermoelectric, industrial, stockwater, fire suppression, snow-making, etc.). Can be one or more uses assigned per site. Preference is to list the uses without abbrieviations, separated by commas.
